# 💻 Alura Plus

Projeto desenvolvido durante o curso de **HTML e CSS** da [Alura](https://www.alura.com.br/), com o objetivo de praticar a criação de layouts modernos e responsivos.
A proposta do site é divulgar o **Alura Plus**, uma plataforma fictícia de streaming com os conteúdos educacionais da Alura.

---

## 🚀 Tecnologias utilizadas

Este projeto foi construído com as seguintes tecnologias:

- ✅ **HTML5**
- 🎨 **CSS3**

---

## ✨ Funcionalidades

- 📺 Página de **divulgação** de um serviço de streaming fictício  
- 🧩 Layout moderno com **Flexbox** e **Grid**
- 🖌️ Tipografia personalizada com **Google Fonts**
- 🎛️ Paleta de cores com **variáveis CSS**

## 🧠 Aprendizados

Durante a construção deste projeto, foram praticados conceitos fundamentais como:

- Estruturação de páginas com **HTML semântico**
- Estilização com **CSS moderno**
- Criação de layouts com **Flexbox** e **Grid**
- Aplicação de **Google Fonts**
- Uso de **variáveis CSS**
- Organização de pastas e código limpo

## 💻 Como rodar localmente

Para visualizar o projeto em sua máquina:

1. Clone o repositório:

```bash
git clone https://github.com/pedrofaleirosss/aluraplus.git
```

2. Acesse a pasta do projeto:
   
```bash
cd aluraplus
```

3. Abra o arquivo index.html com seu navegador, ou utilize uma extensão como Live Server no VS Code.
